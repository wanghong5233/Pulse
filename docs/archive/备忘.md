# OfferPilot 运行备忘

## 说明

- 服务运行在 **WSL** 里，不依赖 Cursor。
- 关闭 Cursor **不会**停止服务。
- 只要 **不手动关闭 WSL、不重启电脑**，服务会一直运行。
- 这些命令都可以在 **PowerShell 任意目录** 直接执行。

## 常用命令

> 必须用 `offerpilotctl.sh`，它会走 `start_backend.sh` 自动加载 `.env`。  
> 不要再直接裸跑 `uvicorn`，否则可能连错数据库导致拿不到画像。

### 检查状态

```powershell
wsl -e bash -lc "bash /mnt/e/0找工作/0大模型全栈知识库/OfferPilot/scripts/offerpilotctl.sh status"
```

### 一键启动

```powershell
wsl -e bash -lc "bash /mnt/e/0找工作/0大模型全栈知识库/OfferPilot/scripts/offerpilotctl.sh start"
```

### 一键停止

```powershell
wsl -e bash -lc "bash /mnt/e/0找工作/0大模型全栈知识库/OfferPilot/scripts/offerpilotctl.sh stop"
```

### 一键重启

```powershell
wsl -e bash -lc "bash /mnt/e/0找工作/0大模型全栈知识库/OfferPilot/scripts/offerpilotctl.sh restart"
```

### 查看后端日志

```powershell
wsl -e bash -lc "bash /mnt/e/0找工作/0大模型全栈知识库/OfferPilot/scripts/offerpilotctl.sh logs 80"
```

### 查看 Guard 业务日志

```powershell
wsl -e bash -lc "bash /mnt/e/0找工作/0大模型全栈知识库/OfferPilot/scripts/offerpilotctl.sh guard-logs 120"
```

### 检查 default 画像是否可读

```powershell
wsl -e bash -lc "bash /mnt/e/0找工作/0大模型全栈知识库/OfferPilot/scripts/offerpilotctl.sh profile"
```

## 当前行为

- 夜间非活跃时段：Guard 可能处于 `sleeping: true`
- 休眠时浏览器会关闭，属于正常资源释放
- 到活跃时段后会自动唤醒
- 工作日高峰时段会自动主动打招呼
